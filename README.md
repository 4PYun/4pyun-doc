# P云开放平台

> [!TIP|label:平台简介]
> P云是服务全国广大停车场系统公司（系统厂商）的互联网开放平台，为系统厂商赋能解决物业（停车场）需求，提升系统厂商核心竞争力！

## 平台能力

### I 停车平台
- 扫码支付
- 无感支付
- 固定车续费
- 停车优惠
- 会员积分
- 集中管控
- 预约停车
- 城市平台上报
- 电子发票

[接入指引](parking/guideline/index.html)

### II 聚合支付
- 为合作伙伴提供快速接入移动端支付能力！

[接入指引](openapi/adverting.html)

### III 广告联盟
平台依托庞大停车行业数据实现车主出行垂直领域精准投放，实现流量变现。

[接入指引](openapi/api/adverting-jsapi.html)
